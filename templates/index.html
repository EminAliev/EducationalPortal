Программирование на Python

В этом курсе по программированию на языке Python вы познакомитесь с базовыми понятиями программирования.

Алгоритмы:Теория и практика. Методы

В курсе будут подробно разобраны базовые алгоритмические методы: жадные алгоритмы, метод «разделяй и властвуй», динамическое программирование. Для всех алгоритмов будут математически строго доказаны корректность и оценки на время работы.

Дискретная математика

За два месяца, на которые рассчитан курс, слушатели ознакомятся с некоторыми разделами дискретной математики. Это линейная алгебра и комбинаторика.
